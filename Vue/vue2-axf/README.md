  ## 启动项目

  ``` bash
 # 安装依赖
 npm install
 
 # 运行项目自： localhost:8080
 npm run dev
 # 还需要运行json-server
 npm run db
 
 # 打包项目
 npm run build
 
 # build for production and view the bundle analyzer report
 npm run build --report
  ```

 初始化`mock`数据通过在项目目录下执行： `node mock.js`

登陆页中输入不存在的手机默认就是注册，可以输入`18600805411`登陆体验
